Theming

Theming can be done overriding Sass variables and custom CSS as well as in the bootstrap.

Animations

The sidebar comes with two predefined animations:

Animation Description
.sidebar-fade Fade in and fade out effects.
.sidebar-slide-left Slide from left to right.
